Report Generation and Storytelling

Research has analyzed provenance data to automatically generate summary reports of an analysis session. Since a user’s interaction history can be long and varied, “chunking” to reduce the complexity of the history log, and “authoring” to generate reports and stories to reflect the relevant of the analysis are two common challenges. For example, Click2Annotate uses low-level tasks results to create insight summaries with automated annotations. Similarly, InsideInsights produces automated data-driven reports that allow the analyst to edit and structure insights into hierarchical views. Chart Constellations generates summary insights from observations in a collaborative system. Lastly, CLUE supports a user to directly interact with the history (provenance) graph to generate a story from the user’s analysis history.
